# Roadmap (4 iterações)

## Iteração 1 (fundação)
- DB + index + API /jobs
- 1 fonte estável + normalização
- UI básica no site (pode ser depois)

## Iteração 2 (qualidade)
- Dedup melhorada
- Enrichment por tags/seniority (IA opcional)
- Mais 2 fontes

## Iteração 3 (distribuição)
- Bot Telegram (post + digest)
- Painel simples para habilitar/desabilitar fontes

## Iteração 4 (refino)
- Ranking melhor
- Observabilidade (métricas por fonte)
- Alertas (inicial)
